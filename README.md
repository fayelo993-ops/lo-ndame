# LO NDAME — Maison de Design 🛍️

Boutique e-commerce de décoration intérieure basée à **Dakar, Sénégal**.

## 🌐 Site en ligne
👉 [teal-florentine-8a8e72.netlify.app](https://teal-florentine-8a8e72.netlify.app)

## 📌 Description
Lo Ndame est une boutique en ligne de design d'intérieur proposant des objets de décoration soigneusement sélectionnés. Le site permet aux utilisateurs de parcourir le catalogue, filtrer les produits par catégorie, et ajouter des articles au panier.

## ✨ Fonctionnalités
- 🛒 Catalogue de 8 produits avec photos
- 🔍 Filtres par catégorie (Luminaires, Mobilier, Accessoires)
- ↕️ Tri par prix et par nom
- 🛍️ Panier dynamique avec gestion des quantités
- 📧 Formulaire de newsletter
- 📱 Design responsive (mobile & desktop)
- 💰 Prix en FCFA

## 🛠️ Technologies utilisées
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 Déploiement
Le site est déployé sur **Netlify** via Netlify Drop.
